Links

Com tag <a> é possível adicionar um link no documento. Esse link poderá levar para um outro endereço ou um arquivo HTML dentro do documento, tal como tá sendo usado neste site.

É capaz de ter links que não levam a nenhum endereço. Mas outros levam a uma URL(endereço). Utilizando o atributo href, de valor "URL" na tag de abertura <a href> é possível colocar um link.
